2.8 Prática - Principais marcações para o corpo

o "<!DOCTYPE html>" serve para identificar o documento como html, para que os browsers interpretarem corretamente

o "<html>" é a tag que envolve o corpo e o cabeçalho do código

"<head>" serve para administrar os metadados, e as partes do código que não são visíveis para os usuários

"<body>" é a tag que envolve tudo que será visto e interagível pelo usuário
